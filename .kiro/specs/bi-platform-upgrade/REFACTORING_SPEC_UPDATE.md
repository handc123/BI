# 仪表板与可视化功能整合 - 规格文档更新

## 更新日期
2026-01-20

## 更新概述

根据用户需求,将"仪表板与可视化功能整合"的需求正式纳入BI平台升级规格文档。此次更新包括需求文档、设计文档和任务列表的完整更新。

## 更新内容

### 1. 需求文档更新 (requirements.md)

**新增需求**: 需求8 - 仪表板与可视化功能整合

**验收标准** (10条):
1. 统一的工具栏,包含图表、筛选器、富文本、媒体、Tab和更多组件
2. 图表库选择对话框,展示所有可用图表类型
3. 实时预览和基础样式配置面板
4. 支持设置标题、颜色、内边距、圆角、边框等样式属性
5. 发布按钮将图表添加到画布
6. 选中组件时显示详细配置选项
7. 保存仪表板时保存所有组件配置
8. 保留后端Visualization API
9. 只显示仪表板管理入口,移除可视化管理菜单
10. 支持拖拽式布局和网格对齐

### 2. 设计文档更新 (design.md)

**新增章节**: 仪表板与可视化功能整合设计

**包含内容**:

#### 设计目标
- 将可视化管理和仪表板管理合并为统一模块
- 提高用户体验和工作效率

#### 架构变更
- **前端**: 移除独立可视化管理页面,新增图表库组件
- **后端**: 保留Visualization API以支持内部使用

#### 用户界面设计
- 仪表板设计器主界面布局
- 图表库选择对话框布局

#### 图表类型定义
支持10大类图表类型:
1. 仪表盘图表 (3种)
2. 基础柱状图 (4种)
3. 折线图 (4种)
4. 饼图 (3种)
5. 散点图 (2种)
6. 雷达图 (1种)
7. 漏斗图 (1种)
8. 表格 (2种)
9. 指标卡 (3种)
10. 查询控件 (8种)

#### 数据模型变更
- Dashboard数据结构增强,支持内嵌组件配置
- 组件配置包含basicStyle、chartStyle、axis等

#### 组件接口
- BiChartLibrary组件接口定义
- BiDashboardDesigner组件增强

#### 工作流程
- 创建图表流程 (10步)
- 编辑图表流程 (5步)

#### 菜单配置变更
- 删除可视化管理菜单SQL
- 更新仪表板管理菜单权限SQL

#### 迁移策略
- 现有数据迁移方案
- 向后兼容保证

#### 实施计划
4个阶段,8-12个工作日:
- 阶段1: 图表库组件开发 (3-4天)
- 阶段2: 仪表板设计器增强 (2-3天)
- 阶段3: 功能迁移和清理 (1-2天)
- 阶段4: 测试和优化 (2-3天)

#### 风险和挑战
- 用户习惯变更
- 现有数据兼容性
- 性能影响
- 复杂度增加

#### 成功标准
- 完成所有图表创建和配置
- 不再需要独立可视化管理页面
- 现有功能继续正常工作
- 图表创建流程时间减少50%
- 用户满意度提升
