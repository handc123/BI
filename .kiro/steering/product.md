# 产品概述

## IRAS - 智能监管分析系统

IRAS 是一个基于若依框架构建的综合性监管分析和数据质量监控系统。系统为金融监管合规提供实时监控和分析能力。

## 核心模块

- **数据质量看板**：监控跨机构的数据质量指标、规则合规性和改进率
- **转换质量看板**：跟踪数据转换质量、表级质量检查和失败分析
- **指标质量看板**：监控指标偏差、排名和统计分析
- **全局总览**：提供信贷风险、关键业务指标、房地产金融和不良资产的综合监控仪表板

## 主要功能

- 多维度数据质量监控和排名
- 机构和区域质量对比
- 基于规则的质量检查，详细的失败追踪
- 使用 ECharts 的实时仪表板可视化
- 基于角色的访问控制和权限管理
- 支持部门层级的多租户架构
- 通过 Springdoc/OpenAPI 提供 API 文档

## 目标用户

需要全面数据质量监控和监管分析能力的金融机构、监管机构和合规团队。
