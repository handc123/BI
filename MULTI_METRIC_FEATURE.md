# 多指标详情和血缘图谱功能说明

## 📊 功能概述

当图表配置了**多个指标**时（如折线图、柱状图等），点击图表会显示所有指标的详情和血缘关系。

---

## 🎯 主要特性

### 1. 自动识别所有指标
系统会自动匹配图表中配置的所有指标字段：

```
图表配置示例：
- 不良贷款率 (non_performing_loan_ratio)
- 关注类不良贷款比率 (concern_non_performing_loan_ratio)
- 逾期90天以上贷款比率 (overdue_90days_loan_ratio)
```

### 2. 左侧指标列表侧边栏
显示所有匹配到的指标：
- 指标名称（用户可读）
- 指标编码（技术标识）
- 当前选中标记

### 3. 点击切换查看
- 点击列表中的任意指标，右侧内容区会切换显示该指标的详情
- 当前选中指标会有高亮标记

### 4. 三个标签页
每个指标都可以查看：
- **指标规范**：业务定义、技术公式、计算逻辑
- **血缘图谱**：上游和下游血缘关系
- **数据查询**：查询和导出数据

---

## 🔄 工作流程

```
用户点击图表
    ↓
获取图表配置的所有指标字段
    ↓
根据字段名自动匹配指标元数据
    ↓
匹配成功？
    ├─ 是 → 打开多指标详情对话框
    │        ├─ 左侧：指标列表（可点击切换）
    │        └─ 右侧：当前选中指标的详情
    └─ 否 → 显示警告信息
```

## 🔧 技术实现

### 前端组件修改

#### 1. MetricDetailDialog/index.vue
**新增Props**:
```javascript
props: {
  metricId: [Number, String],      // 单个指标ID（向后兼容）
  metricIds: Array,                 // 多个指标ID（新增）
  metricList: Array                 // 指标列表信息（新增）
}
```

**新增数据**:
```javascript
data() {
  return {
    currentMetricId: null,  // 当前选中的指标ID
    metricInfo: {}          // 当前指标信息
  }
}
```

**新增方法**:
```javascript
// 初始化指标（单指标/多指标）
initializeMetrics()

// 切换指标
handleMetricChange(metricId)
```

#### 2. ChartWidget/index.vue
**修改方法**: `handleChartClick()`
- 遍历所有指标字段
- 匹配每个字段的指标元数据
- 触发 `multi-metric-click` 事件

**新增事件**:
```javascript
this.$emit('multi-metric-click', matchedMetrics, metricList)
```

#### 3. view.vue
**新增数据**:
```javascript
data() {
  return {
    selectedMetricIds: [],   // 多指标ID数组
    selectedMetricList: []   // 指标列表信息
  }
}
```

**新增方法**:
```javascript
handleMultiMetricClick(metricIds, metricList)
```

## 📝 使用示例

### 场景1：双指标折线图

**图表配置**:
```javascript
{
  metrics: [
    { field: 'non_performing_loan_ratio', label: '不良贷款率' },
    { field: 'concern_non_performing_loan_ratio', label: '关注类不良贷款比率' }
  ]
}
```

**点击后的效果**:
1. 打开多指标对话框
2. 左侧显示2个指标
3. 默认选中第一个（不良贷款率）
4. 可点击切换查看第二个

### 场景2：多指标柱状图

**图表配置**:
```javascript
{
  metrics: [
    { field: 'non_performing_loan_ratio', label: '不良贷款率' },
    { field: 'concern_non_performing_loan_ratio', label: '关注类比率' },
    { field: 'overdue_90days_loan_ratio', label: '逾期90天比率' }
  ]
}
```

**点击后的效果**:
1. 打开多指标对话框
2. 左侧显示3个指标
3. 点击任意指标查看详情和血缘

## ✅ 兼容性说明

### 向后兼容
- 单指标图表仍然使用 `metricId` 参数
- 原有功能不受影响
- 新增功能完全向后兼容

### 判断逻辑
```javascript
computed: {
  isMultiMetric() {
    return this.metricIds && this.metricIds.length > 1
  }
}
```

---

## 🐛 注意事项

1. **指标匹配**: 图表字段名必须与数据库中 `metric_code` 完全一致
2. **数据加载**: 切换指标时会自动刷新当前标签页内容
3. **性能考虑**: 一次性加载所有指标ID，但按需加载详情
